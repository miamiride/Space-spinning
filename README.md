# Scratch Glide Demo – Flying Sprite

## Overview
This is a simple Scratch project that demonstrates basic motion and events.
When the green flag is clicked, the sprite glides from point A → B → C → D and displays a message:  
"Hey look I am flying!"

## What It Demonstrates
- Events: `when green flag clicked`
- Motion: `go to x:y`, `glide _ secs to x:y`
- Looks: `say "Hey look I am flying!"`

## How to Run
1. Open the project in Scratch.
2. Click the green flag.
3. Watch the sprite glide through the points and show the message.

## Learning Goal
This project is meant as a beginner-friendly example to show how motion and events work together in Scratch.
